# UI Implementation Rules (shadcn-first)

## UI 部品

- 素の <button>/<select>/<input> の直書きより、原則 shadcn（Button/Select/Input/Card 等）を使う
- 共通状態 UI は shared に寄せる（PageSkeleton / EmptyState / ErrorState）

## 画面実装

- ViewState（union）で描画を switch する（idle/loading/success/error 等）
- エラー表示は `toErrorMessage(err: unknown)` 等の共通関数で統一（any 禁止）

## 禁止

- UI から API を直接呼ばない（PageModel だけを見る）
- UI で contract を `as` キャストで誤魔化さない
